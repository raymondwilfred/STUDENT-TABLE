# STUDENT-TABLE
Pro-UI Student Data Dashboard
A clean, responsive data table interface designed with a focus on modern aesthetics and information hierarchy.

Design Features:

Zebra Striping: Implemented :nth-child(even) CSS logic to enhance row readability.

Modern Aesthetic: Utilized linear gradients for the background and soft box-shadow depth for the table container.

Rounded UI: Applied border-radius to the table and header to break away from the "boxy" look of standard HTML tables.

Information Architecture: Used bold typography and contrasting background colors for the table headers to separate categories clearly.
